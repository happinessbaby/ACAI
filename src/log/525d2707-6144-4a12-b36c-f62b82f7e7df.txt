[0m[32;1m[1;3mSome common techniques used for data analysis include:

1. Text Analysis: This technique involves discovering patterns in large data sets using databases or data mining tools. It helps transform raw data into business information and can be used for extracting and examining data, deriving patterns, and interpreting the data.

2. Statistical Analysis: Statistical analysis involves analyzing data to understand "what happened" by using past data in the form of dashboards. It includes collection, analysis, interpretation, presentation, and modeling of data. Descriptive analysis and inferential analysis are two categories of statistical analysis.

3. Diagnostic Analysis: Diagnostic analysis aims to find the cause of a particular outcome by analyzing insights found in statistical analysis. It helps identify behavior patterns in data and can be useful in identifying similar patterns for new problems or situations.

4. Predictive Analysis: Predictive analysis focuses on predicting future outcomes based on previous data. It involves making predictions about future outcomes based on current or past data. Forecasting is an example of predictive analysis, but it requires detailed information and analysis.

5. Prescriptive Analysis: Prescriptive analysis combines insights from previous analyses to determine the best course of action for a current problem or decision. It goes beyond predictive and descriptive analysis to improve data performance and make data-driven decisions.

These are just a few examples of common techniques used for data analysis. Each technique has its own purpose and can be applied in different scenarios depending on the nature of the data and the desired outcomes.
